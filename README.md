#  Anime Next Watch Generator

## Project Overview

**Anime Next Watch Generator** is a web application designed to help users quickly decide what anime to watch next. With thousands of anime available, users often face decision fatigue. This project solves that problem by providing smart, quick, and engaging anime recommendations.

This milestone focuses on **planning, structure, and clarity of idea**, ensuring the project is feasible and scalable for future development.

---

## Purpose

The goal of this project is to:

* Help users discover new anime easily
* Reduce time spent searching for what to watch
* Provide a clean and interactive recommendation experience
* Lay the foundation for advanced features like filtering, search, and personalization

---

## API Used

The project will use an external Anime API such as:

* **Jikan API (MyAnimeList unofficial API)**
   https://api.jikan.moe/

### Why this API?

* Free and publicly accessible
* Large anime database
* Provides detailed information like:

  * Title
  * Genre
  * Rating
  * Episodes
  * Synopsis
  * Images


*  **Filtering**

  * Filter by genre (Action, Romance, Comedy, etc.)
  * Filter by rating

* 📊 **Sorting**

  * Sort by popularity
  * Sort by rating
  * Sort alphabetically

*  **Watchlist Feature**

  * Save favorite anime

*  **Next Recommendation Button**

  * Generate new suggestions instantly

## ⚙️ How the Project Will Work

1. User opens the website
2. Clicks on “Generate Anime”
3. Application sends a request to the Anime API
4. API returns data in JSON format
5. Data is processed and displayed on the UI
